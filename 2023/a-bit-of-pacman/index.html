A bit of pacman⌗

Pacman⌗

A package manager for Arch Linux and its variants.

Installing packages⌗ 

sudo pacman -S <package>

Removing packages⌗ 

sudo pacman -R <package>

Updating packages⌗ 

sudo pacman -Syu

Searching packages⌗ 

sudo pacman -Ss <package>

Querying packages⌗ 

sudo pacman -Qi <package>

Querying files⌗ 

sudo pacman -Ql <package>

Extras⌗

Database lock file⌗ 

sudo rm /var/lib/pacman/db.lck
